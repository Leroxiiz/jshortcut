Introduction

JShortcut is a Java package and JNI library to allow Java programs to create and edit shortcuts (ShellLinks) under Windows.

Home Page

Go to the JShortcut home page to download the latest version of JShortcut.

Features

  • Licensed under GPL, freely redistributable with source.
  • Can be made to work when part of a self-extracting JAR file.

Documentation

The javadoc-generated API Reference provides detailed documentation on the packages and classes used in JShortcut (available after you have unpacked the source).

Installing JShortcut

JShortcut is delivered in a self-extracting JAR file. You must have Java running on your system in order to extract it.

To install JShortcut from the installation JAR file (.jar extension), double-click on the JAR file, or run the command

java -jar jshortcut-N_N_Njar
This will open a dialog allowing you to specify a destination directory, and it will create a directory called jshortcut-N_N_N in the specified directory, where N_N_N is replaced by the JShortcut version number.

Unpacking the JShortcut Source

If you want to build JShortcut yourself, or just look at the source code, you will have to unpack the source files from the src.zip archive file with the commands

cd jshortcut-N_N_N
unzip src.zip
This will create a src directory with all of the Java, C++ other source files, as well as adding a few other files (such as a Makefile) to your jshortcut directory.

If you want to look at the Javadoc-generated API documentation for the Java classes, you must first unpack those html files from the apidoc.zip archive file with the commands

cd jshortcut-N_N_N
unzip apidoc.zip
This will create the directory doc/api with all of the API documentation in it. Alternatively, you can unpack src.zip and then run the command
make doc
to run javadoc and create the API documents from the source files.
